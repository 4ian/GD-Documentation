void gd::PlatformExtension::CloneExtension ( const std::string &  platformName,
const std::string &  extensionName,
bool  stripFunctionsNameAndCodeGeneration = true 
)

Clone the extension of another platform.

This can be used when you want a platform to conform to another.
It is still possible to make some changes after the cloning by using getter methods.
See also gd::PlatformExtension::StripUnimplementedInstructionsAndExpressions.

Parameters
platformNameThe name of the platform in which the source extension must be searched for.
extensionNameThe name of the source extension to be copied.
stripFunctionsNameAndCodeGenerationIf set to true, all functions names and code generator of all instructions/expression/events will be removed.
