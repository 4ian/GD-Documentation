Detailed Description

Property for wxPropertyGrid displaying its value with a button triggering a file selector.

Compared to wxFileProperty, this class allows to use relative filenames: If a base path is specified ( property->SetAttribute(wxPG_FILE_SHOW_RELATIVE_PATH, myBasePath) ), the value returned by the file selector will be relative to this path.

Supports wxPG_FILE_WILDCARD, wxPG_FILE_SHOW_RELATIVE_PATH, wxPG_FILE_INITIAL_PATH, wxPG_FILE_DIALOG_TITLE attributes.
