Detailed Description

Singleton class which can be used by the IDE and the editors to access to a (fake) clipboard.

All objects stored in this clipboard are stored by copy. Getter returns a copy of the stored objects, and can only be called if "Has" method returns true (otherwise, behavior is undefined).
