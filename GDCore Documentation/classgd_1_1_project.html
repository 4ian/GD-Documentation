std::unique_ptr< gd::Behavior > gd::Project::CreateBehavior ( const gd::String &  type,

)

Create a behavior of the given type.

Note
A project can use more than one platform. In this case, the first platform supporting the behavior is used, unless platformName argument is not empty.
It is assumed that each platform provides an equivalent behavior.
Parameters
projectThe project for which the object must be created.
typeThe type of the behavior
platformNameThe name of the platform to be used. If empty, the first platform supporting the object is used.
