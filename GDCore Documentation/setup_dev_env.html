Setting up the development environment

If you didn't already downloaded GDevelop, get and extract the source from GitHub.

Follow these steps to be able to compile GDevelop, it's super easy:

Windows

  1. (Windows) Install TDM-GCC compiler
  2. (All) Install CMake & launch the build

GNU/Linux

  1. (Linux) Install development files
  2. (All) Install CMake & launch the build

OS X

  1. (Mac OS X) Install development tools
  2. (All) Install CMake & launch the build

See the recommended tools and conventions for working on GDevelop on this page: Recommended tools and advices to work with GD
