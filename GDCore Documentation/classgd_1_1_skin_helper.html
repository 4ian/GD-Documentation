Detailed Description

Provide tool functions to easily apply the current skin or theme to controls supporting it.

The skin are loaded from the configuration stored thanks to wxConfigBase. If you use a control supported by one of ApplyCurrentSkin function, you should call it on your control after it has been created.

Get an icon for the interface.

The icon is searched in res/icons_[skinName]/[iconName][size].png with 'SkinName' being the name of the selected skin for the ribbon icons. If no such directory or file exists, res/icons_default/[iconName].png is used.

Icons are cached after being loaded.
