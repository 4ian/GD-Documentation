(Linux) Install development files

Download and extract Game Develop for Linux

If you didn't already downloaded Game Develop for Linux, do it now from the official website.

Then extract all the files inside the GameDevelop folder inside the *Binaries/Output/Release_Linux/ * folder of the SDK. ( You should have files like libGDCore.so now present in Binaries/Output/Release_Linux )

Download, (build) and install libraries

Game Develop is compiled with gcc under Linux.
So as to prevent incompatibilities between the compiler ( and the standard C++ library provided with ) used by Game Develop and the compiler used by the extensions, Game Develop require the extensions and the platforms to use the same version of gcc.

SFML

Game Develop may uses some specific version of SFML library. You can download a precompiled package for linux here: http://www.compilgames.net/code/GameDevelopSDK/SFMLlinux.7z
Just extract the archive inside the folder ExtLibs (located at the root of the SDK).

Boost

Boost version used by Game Develop can be downloaded here: http://www.compilgames.net/code/GameDevelopSDK/boost_1_55_0.7z
Just extract the archive inside the folder ExtLibs (located at the root of the SDK).

wxWidgets

wxWidgets version used by Game Develop can be downloaded here: http://www.compilgames.net/code/GameDevelopSDK/wxWidgetslinux.7z
First extract the archive inside the folder ExtLibs.
You then have to build and install the library: Open a terminal, go to the ExtLibs/wxWidgets directory and follow the classical configure/make/make install process:

./configure --enable-ribbon
make
sudo make install

Note that you need to have GTK+ 2.0 or GTK+ 3.0 development file installed (in your package manager, search for the libgtk2.0-dev or libgtk-3-dev package and install it).

Install CMake

You'll need CMake to build the extensions: See more on the page on how to install and use CMake.
