(Linux) Install development files

Download and extract GDevelop for Linux

If you didn't already downloaded GDevelop for Linux, do it now from the official website.

Then extract all the files inside the GameDevelop folder inside the *Binaries/Output/Release_Linux/ * folder of the SDK. ( You should have files like libGDCore.so now present in Binaries/Output/Release_Linux )

Download, (build) and install libraries

GDevelop is compiled with gcc under Linux.
So as to prevent incompatibilities between the compiler ( and the standard C++ library provided with ) used by GDevelop and the compiler used by the extensions, GDevelop require the extensions and the platforms to use the same version of gcc.

SFML

GDevelop may uses some specific version of SFML library. You can download the version used here: http://www.compilgames.net/code/GameDevelopSDK/SFMLlinux.7z

The archive contains a precompiled version of SFML, compiled for Ubuntu, and only for the version for which GDevelop is distributed officially:

  • If you compile GD on this version of Ubuntu, just extract the archive inside the folder ExtLibs.
  • If you compile GD for another version of Ubuntu or a different distro, extract the archive inside the folder ExtLibs, and recompile SFML using CMake into a folder called build-linux. It's easy, just follow this tutorial. Again, be sure to build the binaries into ExtLibs/SFML/build-linux. If you have difficulties, read this page: (Linux) Manually build SFML, it details the dependencies you need to install.

Boost

Boost version used by GDevelop can be downloaded here: http://www.compilgames.net/code/GameDevelopSDK/boost_1_55_0.7z
Just extract the archive inside the folder ExtLibs (located at the root of the SDK).

wxWidgets

wxWidgets version used by GDevelop can be downloaded here: https://sourceforge.net/projects/wxwindows/files/3.0.2/wxWidgets-3.0.2.tar.bz2
First extract the archive inside the folder ExtLibs.

Be sure to have GTK+ 3.0 and WebKitGTK development files installed:

sudo apt-get install libgtk-3-dev
sudo apt-get install libwebkitgtk-3.0-dev

You then have to build and install the library: Open a terminal, go to the ExtLibs/wxWidgets directory and follow the classical configure/make/make install process:

./configure --enable-ribbon --enable-webview
make
sudo make install

Install CMake

You'll need CMake to build GDevelop: See more on (All) Install and use Cmake.
