Detailed Description

Tool dialog used by LayersEditorPanel to ask the user what to do of instances which are on a layer being deleted.

Calling ShowModal() on this class will return:

  • 0 to cancel everything
  • 1 to delete the instances
  • 2 to move them on another layer
