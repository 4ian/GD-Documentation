Detailed Description

Describe a GUI element drawn on a LayoutEditorCanvas, like a resize button or a context button.

These buttons are most of time created by the specialization of the LayoutEditorCanvas, during rendering. When LayoutEditorCanvas detects a click on a button, it calls OnGuiElementHovered/Pressed/Released.
