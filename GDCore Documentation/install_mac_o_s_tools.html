(Mac OS X) Install development tools

Make sure that you have Apple Developer Tools installed (if you have Xcode and git, that should be the case).

Install development tools

The simplest way of installing dependencies required by GDevelop is to use Homebrew. Install it and install these packages, using the terminal:

brew install cmake
brew install p7zip
brew install pkgconfig
brew install wxwidgets

If you want to generate the documentation and translations, install Doxygen and Gettext:

brew install doxygen
brew install gettext

Launch compilation

You should be able to compile GD using CMake. Go with a terminal to the GD source folder:

cd /path/to/GD
mkdir build && cd build
cmake ../..
make -j4

More information about compilation here: (All) Install CMake & launch the build
