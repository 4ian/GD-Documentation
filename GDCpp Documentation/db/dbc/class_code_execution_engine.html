Detailed Description

Wrapper allowing to load a dynamic library and launch a specific function.

RuntimeScene uses this class to launch compiled events. In particular, the signature of the function of the dynamic library must be :

void functionName(RuntimeContext *);

: This class is unecessarily complicated.
