Detailed Description

Class containing helper functions for using CodeCompiler See CreateSceneEventsCompilationTask to automatically launch a compilation for a scene from the editor.

3 classes represents thing that can be compiled: Scenes, as they provide the context to the events they own. External source files, which can be used through C++ code events. External events, which can be also compiled separately.
