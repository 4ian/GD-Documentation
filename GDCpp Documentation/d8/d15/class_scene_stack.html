◆ Step()

bool SceneStack::Step ( )

Execute one step of the game.

RuntimeScene::RenderAndStep is called on the current scene. If a scene change was requested, the stack is updated.

This method is typically called in a loop until it returns false.

Returns
false if game must be stopped.
