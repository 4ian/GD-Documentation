Detailed Description

Base class for C++ extensions. Extensions can provide :

  • Static functions ( e.g. GetActionFunctionPtr ).
  • Objects functions ( e.g. GetObjectActionFunctionPtr )
  • New objects, which have a type. The new objects creations/destructions functions are provided by the extension.
  • More information when compiled for the IDE.
