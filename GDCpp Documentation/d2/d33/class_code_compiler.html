Detailed Description

C++ Code compiler This class uses a thread to launch Clang compiler according to the task added using AddTask. Specific functions are available for preventing the compiler to start a new task involving a specific scene.

void CodeCompiler::AllowMultithread ( bool  allow = true,
unsigned int  maxThread = 3 
)

Set if CodeCompiler is allowed to launch more than one thread.

For example, if a task is requested twice in a short time, CodeCompiler can launch a 2nd thread to do the work and discard the result of the 1st as it is useless. If multithread is disabled or if maxThread count is reached, then the second task is added to pending tasks.
