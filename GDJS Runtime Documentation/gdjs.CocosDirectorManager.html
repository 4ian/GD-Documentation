Class: CocosDirectorManager

CocosDirectorManager

new gdjs.CocosDirectorManager(The)

cocos-renderers/cocos-director-manager.js, line 7

Encapsulate the Cocos2D director singleton, and makes sure that Cocos scenes that are pushed/popped from it are done according to the director needs (always a scene living, different call to push the 1st scene).

Name Type Description
The Object

Cocos2D-JS to use. If not specified, cc.director will be used.
