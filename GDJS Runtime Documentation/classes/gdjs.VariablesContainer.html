    getFromIndex

    (
    • id
    )

    Defined in ../Runtime/variablescontainer.js:119

    Get a variable using its index.<br> The index of a variable is its index in the data passed to initFrom.<br>

    This method is generally used by events generated code to increase lookup speed for variables.<br> If you're unsure about how to use this method, prefer to use get.

    Parameters:

    • id Number

      The variable index

    Returns:

    The specified variable. If not found, an empty variable is added to the container, but it should not happen.

    initFrom

    (
    • data
    • keepOldVariables
    )

    Defined in ../Runtime/variablescontainer.js:24

    Initialize variables from a container data.<br> If keepOldVariables is set to false ( by default ), all already existing variables will be erased, but the new variables will be accessible thanks to getFromIndex. <br> if keepOldVariables is set to true, already existing variables won't be erased and will be still accessible thanks to getFromIndex.

    Parameters:

    • data Object

      The object containing the variables.

    • keepOldVariables Boolean

      If set to true, already existing variables won't be erased.
