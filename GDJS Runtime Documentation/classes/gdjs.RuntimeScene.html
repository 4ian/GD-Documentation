    _cacheOrClearRemovedInstances

    () private

    Defined in ../Runtime/runtimescene.js:237

    Empty the list of the removed objects:
    When an object is removed from the scene, it is still kept in the _instancesRemoved member of the RuntimeScene.
    This method should be called regularly ( after events or automatisms steps ) so as to clear this list and allows the removed objects to be cached ( or destroyed if the cache is full ).
    The removed objects could not be sent directly to the cache, as events may still be using them after removing them from the scene for example.
