    _transformToGlobal

    (
    • pointX
    • pointY
    • result
    )
    private

    Defined in ../Runtime/spriteruntimeobject.js:487

    Return an array containing the coordinates of the point passed as parameter in world coordinates (as opposed to the object local coordinates).

    Beware: this._animationFrame and this._sprite must not be null!

    All transformations (flipping, scale, rotation) are supported.

    Parameters:

    • pointX Object

      The X position of the point, in object coordinates.

    • pointY Object

      The Y position of the point, in object coordinates.

    • result Object

      Optional array that will be updated with the result (x and y position of the point in global coordinates)

    Returns:

    An array containing the x and y position of the point in global coordinates. If result parameter is defined, nothing is returned by the method.
